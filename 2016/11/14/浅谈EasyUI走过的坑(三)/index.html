闲聊EasyUI以及项目中踩过的坑(三)：EasyUI中零碎问题

经验分享
  • EasyUI
  • jQuery

版权声明：本文为VII-Lin原创文章，转载请注明出处，未经允许不得转载

上篇文章简单介绍了EasyUI中Combotree中众多使用问题，这一篇作为一些零碎知识点的汇总


DataGrid数据表格

下面为DataGrid最基本的构建方式，分别为列、行、数据。

<table class="easyui-datagrid">   
    <thead>   
        <tr>   
            <th data-options="field:'code'">编码</th>   
            <th data-options="field:'name'">名称</th>   
            <th data-options="field:'price'">价格</th>   
        </tr>   
    </thead>   
    <tbody>   
        <tr>   
            <td>001</td><td>name1</td><td>2323</td>   
        </tr>   
        <tr>   
            <td>002</td><td>name2</td><td>4612</td>   
        </tr>   
    </tbody>   
</table>

同时，最常用的加载方式是使用js去创建DataGrid

<table id="dg"></table>

$('#dg').datagrid({    
    url:'datagrid_data.json',    
    columns:[[    
        {field:'code',title:'Code',width:100},    
        {field:'name',title:'Name',width:100},    
        {field:'price',title:'Price',width:100,align:'right'}    
    ]]    
});

常用的一些属性：

method， 一般为POST请求远程数据

url，远程站点地址

pagination，使用分页栏工具

pageNumber，分页时初始化页码

pageSize，分页是初始化页面大小

pageList，默认为[10,20,30,40,50]

queryParams，请求远程时发送参数

TextBox文本框

通过标签创建文本框

<input class="easyui-textbox" data-options="iconCls:'icon-search'" style="width:300px">

也可用js创建

<input id="tb" type="text" style="width:300px">

$('#tb').textbox({    
    buttonText:'Search',    
       iconCls:'icon-man', 
    iconAlign:'left'       
})

常见属性：

width，组件宽度

height，组件高度

type，文本框类型，有text，password两种

multiline，是否允许多行文本框

editable，是否允许直接输入文字

disabled，是否禁用

readonly，是否只读

这里需要注意的是，不论什么控件，当时用了disabled后，是无法获取到其中的value的，一般建议使用readonly，但如果页面确实需要disable的效果，那就使用一个隐藏域来记录标签中的值。

Datebox日期框

日期框（datebox）把可编辑的文本框和下拉日历面板结合起来，用户可以从下拉日历面板中选择日期。

使用标记创建日期框

<input id="dd" type="text" class="easyui-datebox" required="required">

使用js来创建

<input id="dd" type="text">

$('#dd').datebox({
    required:true
});

常用属性

panelWidth，下拉框面板宽度

panelHeight，下拉框面变高度

currentText，显示当天按钮

disabled，是否禁用

formatter，一般用来对日期进行格式化

$.fn.datebox.defaults.formatter = function(date){
    var y = date.getFullYear();
    var m = date.getMonth()+1;
    var d = date.getDate();
    return m+'/'+d+'/'+y;
}

当然啦，常用的控件也不止这几个，但作为后台项目，这几个控件是最为常用的，基本所有的页面都会遇到，这几个控件也就数combotree使用起来麻烦，坑点多。

其实，大多数控件在看官网文档也能用个七七八八了，更多的细节也都是自己真正踩过后，才能牢记的。总之EasyUI是一款非常不错的后台页面框架，企业级项目开发使用也分方便。
